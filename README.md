## How to use

In root `build.gradle`:

```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

In your module/app `build.gradle`:

```gradle
	dependencies {
	        implementation 'com.github.Depau:kotlet:v0.1'
	}
```
